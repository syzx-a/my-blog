# 图片资源目录

此目录用于存放网站所需的图片资源。

## 目录结构

```
images/
├── uploads/          # CMS上传的图片
├── posts/           # 文章配图
├── live/            # 直播封面
├── avatar.jpg       # 个人头像
└── favicon.ico      # 网站图标
```

## 图片规格建议

- **头像**: 400x400px, JPG/PNG
- **文章封面**: 1200x630px, JPG/WebP
- **直播封面**: 1280x720px, JPG/WebP
- **正文配图**: 宽度不超过 1200px

## 图片压缩

建议使用工具压缩图片：

- [TinyPNG](https://tinypng.com/) - 在线压缩
- [Squoosh](https://squoosh.app/) - Google出品
- [ImageOptim](https://imageoptim.com/mac) - Mac工具
